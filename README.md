## 📖 Sobre o Projeto
Este repositório é o meu **Laboratório de Aprendizado**. Aqui, eu documento meu progresso explorando o ecossistema moderno do **PHP**. O objetivo principal é dominar a criação de aplicações robustas utilizando o framework **Laravel** e interfaces administrativas poderosas com **Filament PHP**.

> [!TIP]
> Este projeto é atualizado constantemente à medida que novos conceitos de arquitetura e pacotes são explorados.

---

## 🛠️ Tecnologias e Recursos Explorados

### Back-end & Frameworks
* **[Laravel](https://laravel.com/):** O alicerce das aplicações (Rotas, Eloquent, Middlewares).
* **[Filament PHP](https://filamentphp.com/):** Desenvolvimento acelerado de painéis administrativos e formulários.
* **[MySQL](https://www.mysql.com/):** Modelagem de dados, índices e integridade referencial.

### Front-end & Interatividade
* **TALL Stack:** O combo poderoso de Tailwind, Alpine.js, Laravel e Livewire.
* **Blade Components:** Reutilização de código de interface de forma limpa.
